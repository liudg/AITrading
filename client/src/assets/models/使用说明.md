# 如何添加AI模型图标

## 快速开始

1. **下载图标**
   - 从各AI模型的官网下载logo
   - 或使用截图工具获取清晰的logo图片

2. **准备图标文件**
   - 格式：PNG（推荐透明背景）
   - 尺寸：128x128px 或更高
   - 文件大小：建议小于100KB

3. **放置图标**
   - 将图标文件放在此目录：`client/src/assets/models/`
   - 按以下命名规则命名：

## 文件命名规则

| AI模型 | 文件名 |
|--------|--------|
| DeepSeek | `deepseek.png` |
| 通义千问 | `qwen.png` |
| GPT/ChatGPT | `gpt.png` |
| Claude | `claude.png` |
| Gemini | `gemini.png` |
| LLaMA | `llama.png` |
| Mistral | `mistral.png` |
| 零一万物 | `yi.png` |

## 示例

假设您要添加 DeepSeek 的图标：

```
1. 从 DeepSeek 官网获取 logo
2. 保存为 deepseek.png
3. 将文件放到：client/src/assets/models/deepseek.png
4. 刷新浏览器，图标即可显示
```

## 技术说明

图标会自动：
- 居中显示在 48x48px 的容器中
- 缩放至 40x40px（保持比例）
- 如果加载失败，显示文字缩写
- 支持透明背景
